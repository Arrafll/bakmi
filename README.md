# 🍜 Aplikasi Bakmi — Sistem Pemesanan & Manajemen Restoran

Aplikasi ini adalah sistem terpadu untuk mengelola restoran bakmi, mencakup pemesanan oleh pelanggan dan panel admin untuk operasional harian. Dibangun menggunakan stack modern berbasis server-driven SPA untuk efisiensi dan maintainability.

---

## 🚀 Fitur Utama

### 👤 Customer (Frontend)

* Melihat menu bakmi dan minuman
* Menambahkan pesanan ke keranjang
* Checkout pesanan
* Status pesanan (opsional, jika realtime diimplementasikan)

### 🛠️ Admin Panel (`/admin`)

* Dashboard ringkasan (penjualan, order aktif)
* Manajemen menu (CRUD)
* Manajemen pesanan
* Manajemen user (opsional)
* Laporan sederhana

---

## 🧱 Teknologi yang Digunakan

* Backend: PHP (Laravel)
* Frontend: Vue.js + Inertia.js
* Styling: Tailwind CSS
* Build Tool: Vite
* Database: MySQL
